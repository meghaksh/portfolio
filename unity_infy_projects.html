Augmented Reality Projects

AR Router Instruction Manual

Development Brief

In this application, I used unity3d game engine and Vuforia's object target feature to augment various ports of the router. User do not need to read the physical instruction manual. Also at last I used deep linking concept to merge the application with existing vuforia chalk application.

The company

Infosys, a global leader in technology services & consulting, helps clients in more than 50 countries to create & execute digital transformation strategies.

Purpose of this product

Users can easily configure their newly purchased router(or any other device) without physical presence of technician

My role

Developer

Team Details

Test Platforms

Android, iOS

Difficulties Faced

Detection is not accurate under certain lighting condition. Performance is degraded on lower end mobile devices.

Future Enhancements

Development Process

Gathering Requirements

When I was assigned this project, I had no idea about how Augmented Reality works. I started with the requirement gathering. Below are some of the questions I asked before starting the development.
Who is going to be the prospective users of this application?
Are there any specific tools required?
Is there any delivery deadline?
Is there any specific methodology like agile that we have to follow?
What are all the platforms we have to look into? (Android, iOS, Windows etc)

Proposing Tools

The leads assigned me a task to find available tools for Augmented Reality. I found ARCore, ARKit, Vuforia, and Wikitude are providing comprehensive software development kits for AR. As the requirement was to develop for multiple platforms, I proposed two options that can help to create hybrid application across all the platforms. First Wikitude with Cordova as a base platform and second Vuforia with Unity as a base platform. After evaluating all the pros and cons, we decided to go with Vuforia and Unity.

Using Vuforia

Vuforia provides three main features to create Augmented Reality Experience. For this particular project, we chose Object Target feature to detect a 3D object in the field of view of the camera. There are three steps to use Vuforia in Unity.
1) scan the object using Object Target below good lighting conditions.
2) Upload the scanned .obj file in Vuforia Target Manager and download the database.
3) Import the Vuforia SDK in Unity and use the object target

Using Unity

Vuforia has provided a software development kit for Unity. After importing the generated database from Vuforia in Unity, I created UI and Augmentation at specific locations in multiple Unity scenes to create a meaningful application flow.

Building and testing the application

I built this application on Android and iOS platform. It faced difficulties in detection while testing on mobile devices with lower camera resolution and under lower lighting conditions.
